## EPIC 1 — Item Listing & Management (Owner-Side)
### Goal: Allow users to rent out videogames they receive.
### Includes:
    •	Add a videogame listing (title, description, platform, condition, images)
    •	Set rental price and availability
    •	Edit or deactivate a listing
    •	View rental history of each item
    •	See requests from renters and accept/deny

## EPIC 2 - User Account Management (User-Side)
### Goal: Allow users to manage and view their account's data.
### Includes:
    •	Create an account as a User
    •	Manage profile as a User
    •	Recover an account as a User
    •	View purchase history as a User

## EPIC 3 — Game Search & Rental Requests (Renter-Side)
### Goal: Allow renters to find games cheaply and easily rent them.
### Includes:
    •	Game search (by platform, genre, price range, owner rating, age rating)
    •	Game detail page
    •	Submit rental request
    •	View status of requests
    •	View reservation calendar
    •	Payment preview before checkout
    •	Cancel a request before confirmation

## EPIC 4 — Rental Management System (Core Platform Logic)
### Goal: Handle the lifecycle of a rental.
### Includes:
    •	Automatic conflict detection (game unavailable during selected dates)
    •	Rental approval process
    •	Payment processing
    •	Delivery or handover instructions


## EPIC 5 — Admin Dashboard
### Goal: Keep the platform safe, trustworthy, and compliant through account and partnership oversight.
### Includes:
    •	Manage all users (ban, warn, review profile)
    •	View platform metrics:
        o	Active rentals
        o	Revenue
        o	Most popular games

## EPIC 6 - Moderation 
### Goal: Allow the admin the use of automated systems to ensure safe and legal usage of the app.
### Includes:
    •	Fraud and abuse detection
    •	Support ticket system
    •	Rental completion and auto reminders
    •	Late return notifications
    •	Damage report system
    •	Approve or investigate listings
    •	Manage flagged content
