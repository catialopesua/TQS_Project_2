## EPIC 1 — Item Listing & Management (Owner-Side)
### Goal: Allow users to rent out videogames they receive.
### Includes:
#### • User Story 1.1 - Create an account as a User
        As a user, I want to be able to create an account so my information can be properly saved and managed across devices.
#### • User Story 1.2 - Add a videogame listing
        As a user, I want to be able to add a listing for a videogame (including the title, a description, the platform, its condition and some images) so that other users can see it and rent it.
#### • User Story 1.3 - Set rental price and availability
        As a user, I want to be able to change the price and/or availability of a listing I have done.
#### • User Story 1.4 - Edit or deactivate a listing
        As a user, I want to be able to edit or deactivate a listing I have done.
#### • User Story 1.5 - View rental history of each item
        As a user, I want to be able to view the rental history of each item I have posted a listing for and/or have rented.
#### • User Story 1.6 - See requests from renters and accept/deny
        As a user, I want to be able to see requests from people who want to rent my listed items and be able to accept/deny them.

## EPIC 2 — Game Search & Rental Requests (Renter-Side)
### Goal: Allow renters to find games cheaply and easily rent them.
### Includes:
#### • User Story 2.1 - Game search
    As a user, I want to be able to search for game listings and filter them to my needs (by platform, genre, price range, owner and age rating).
#### • User Story 2.2 - Game detail page
    As a user, I want to see detailed information about a game listing (images, description, platform, condition, owner rating, availability) so I can decide whether or not it is suitable for me.
#### • User Story 2.3 - Submit rental request
    As a user, I want to be able to submit a rental request for a game, choosing dates and confirming the rental conditions, so I can start the rental process.
#### • User Story 2.4 - View status of requests
    As a user, I want to be able to view the status of all my rental requests (pending, accepted, rejected, cancelled), so I can keep track of my rentals.
#### • User Story 2.5 View reservation calendar
    As a user, I want be able to see a calendar showing when a game is available or already booked, so I can choose dates that fit both my and the owner's schedules.
#### • User Story 2.6 - Payment preview before checkout
    As a user, I want to be able to preview the costs of the rental and its special conditions (if any) before confirming, so I know exactly how much I will have to pay.
#### • User Story 2.7 - Cancel a request before confirmation
    As a user, I want to be able to cancel a pending rental request, so I’m not  forced to commit to a rental I no longer want.

## EPIC 3 — Rental Management System (Core Platform Logic)
### Goal: Handle the lifecycle of a rental.
### Includes:
#### • User Story 3.1 - Automatic conflict detection
    As a user, I want to receive a warning if the dates I choose are unavailable after I try to rent a game, so I don’t request a rental that overlaps with an existing booking.
#### • User Story 3.2 - Rental approval process
    As a user, I want to receive requests and be able to approve or reject them, so I can control who rents my games.
#### • User Story 3.3 - Payment processing
    As a user, I want to be able to securely submit payments using supported methods, so I can safely complete the rental proccess.
#### • User Story 3.4 - Delivery or handover instructions
    As a user, I want to be able to specify delivery or handover preferences like the meet-up, the shipping or even a locker pickup, so the exchange can go smoothly.


## EPIC 4 — Admin Dashboard
### Goal: Keep the platform safe, trustworthy, and compliant through account and partnership oversight.
### Includes:
#### • User Story 4.1 - Manage all users
    As an admin, I want to be able to search, review, warn, or ban users so the platform remains safe.
#### • User Story 4.2 - Approve or investigate listings
    As an admin, I want to be able to review listings before they go public, so I can ensure they follow platform rules.
#### • User Story 4.3 - Manage flagged content
    As an admin, I want to be able to see flagged listings, reviews, or messages, so I can remove harmful or inappropriate material.
#### • User Story 4.4 - View platform metrics:
    As an admin, I want to be able to see statistics (such as active rentals, revenue and popular games), so I can understand the performance of the platform and its target audience.

## EPIC 5 - Moderation & System Tools
### Goal: Allow the admin the use of automated systems to ensure safe and legal usage of the app.
### Includes:
#### • User Story 5.1 - Fraud and abuse detection
    As an admin, I want the platform to automatically flag suspicious activity, so I can identify potential fraud early.
#### • User Story 5.2 - Support ticket system
    As a user, I want to be able to open support requests when I need help, so I can smoothly resolve issues with rentals or payments.
#### • User Story 5.3 - Rental completion and auto reminders
    As a user, I want the platform to send me automated reminders (pickup, return, overdue), so I can stay informed throughout the rental process.
#### • User Story 5.4 - Late return notifications
    As a user, I want to be notified when a renter is late on returning my item, so I can take appropriate action.
#### • User Story 5.5 - Damage report system
    As a user, I want to be able to file reports about damaged products with photos and descriptions, so disputes can be handled fairly.
